# pyEdit
Editor de texto criado em Python.

Requisitos:
- Python >= 3.9.0
- PySimpleGUI >= 4.56.0
  
Como instalar os requisitos: 
- Python: https://www.python.org/downloads/
- PySimpleGUI: https://pysimplegui.readthedocs.io/en/latest/
